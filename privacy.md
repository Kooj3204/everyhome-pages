## 1. 수집하는 정보

본 앱은 다음 정보를 수집하거나 처리합니다.

- **6자리 룸 코드 및 WebRTC 시그널링 데이터** (offer/answer SDP, ICE 후보 정보)
  - 사용 목적: 두 기기 사이의 직접(P2P) 연결을 수립하기 위함
  - 저장 위치: Google Firebase Firestore
  - 보관 기간: 룸 생성 후 최대 24시간, 이후 자동 삭제 예정
- **Firebase Cloud Messaging(FCM) 토큰**
  - 사용 목적: (향후 모션 감지 알림 기능 활성화 시) 알림 발송을 위한 기기 식별
  - 저장 위치: Google Firebase Firestore
  - 보관 기간: 사용자가 앱을 삭제하거나 토큰이 만료될 때까지

본 앱은 **사용자의 이름, 이메일, 전화번호, 정확한 위치, 결제 정보 등 식별 가능한 개인정보를 수집하지 않습니다.**

## 2. 영상 및 음성의 처리

영상과 음성 스트림은 WebRTC 기술을 통해 **같은 Wi-Fi 네트워크에 연결된 사용자의 두 기기 사이에서만 직접 전송**되며, 본 앱의 서버 또는 어떠한 외부 서버에도 저장되거나 기록되지 않습니다. 연결이 종료되면 해당 스트림은 즉시 사라집니다.

본 버전은 같은 Wi-Fi(동일 공유기) 환경에서의 P2P 직결만 지원하므로, 영상·음성 스트림 데이터는 사용자의 Wi-Fi 네트워크 외부로 전송되지 않습니다. P2P 연결 수립 과정에서 ICE 후보 정보에 기기의 네트워크 주소(주로 사설 IP)가 일시적으로 포함되어 Firestore에 저장될 수 있으며, 이는 24시간 이내에 삭제됩니다.

## 3. 제3자 제공 및 외부 서비스

본 앱은 수집한 정보를 제3자에게 판매하거나 마케팅 목적으로 제공하지 않습니다. 다만 서비스 운영을 위해 다음 외부 서비스를 이용합니다.

- **Google Firebase (Firestore, Cloud Messaging)**: 룸 코드 교환, 알림 발송에 사용됩니다. Firebase의 처리는 Google의 개인정보처리방침에 따릅니다. (<https://policies.google.com/privacy>)
- **Google STUN 서버 (stun.l.google.com)**: P2P 연결 수립을 위한 네트워크 정보 교환에만 사용됩니다.

## 4. 권한 사용 사유

- **카메라 / 마이크**: 사용자가 직접 시작한 영상·음성 송출에만 사용됩니다.
- **포그라운드 서비스(카메라/마이크)**: 카메라 모드로 사용 중인 기기의 화면이 꺼지거나 다른 앱이 활성화되어도 송출이 끊기지 않도록 하기 위함이며, 작동 중에는 상태바에 항상 알림이 표시됩니다.
- **알림**: Foreground Service 알림과 (향후 활성화 시) 모션 감지 알림을 위함입니다.

## 5. 데이터 암호화

- 전송 중 암호화: 모든 Firebase 통신과 WebRTC 영상·음성 스트림은 TLS/DTLS-SRTP로 암호화되어 전송됩니다.
- 저장 시 암호화: Firestore에 저장되는 시그널링 데이터는 Google Cloud의 표준 저장 시 암호화로 보호됩니다.
