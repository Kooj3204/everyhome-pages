# everyhome (에브리홈)

WebRTC 기반 P2P 홈 보안 카메라 앱.

안 쓰는 스마트폰을 카메라로, 메인 폰을 모니터로 사용해 같은 Wi-Fi 네트워크 안에서 영상을 실시간으로 확인할 수 있습니다. 실시간 스트리밍 영상은 두 기기 사이에서만 P2P로 직통 전송되며 어떠한 중개 서버에도 수집·저장되지 않습니다.

> **녹화 영상 보관 안내**: 사용자가 모션 감지 자동 녹화 또는 수동 녹화 기능을 활성화할 경우, 녹화된 MP4 영상 파일은 **사용자 본인의 구글 드라이브 계정 또는 기기 내부 저장소**에만 안전하게 보관됩니다. 앱 개발자 및 제3자는 해당 영상에 접근할 수 없습니다.
