# everyhome (에브리홈)

WebRTC 기반 P2P 홈 보안 카메라 앱.

안 쓰는 스마트폰을 카메라로, 메인 폰을 모니터로 사용해 같은 Wi-Fi 네트워크 안에서 영상을 실시간으로 확인할 수 있습니다. 영상은 두 기기 사이에서만 P2P로 전송되며 어떠한 서버에도 저장되지 않습니다.
